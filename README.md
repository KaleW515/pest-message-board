# pest-message-board

- 一个简易的网络留言板,基本功能都写好了
- 使用docker快速部署
- 使用websocket进行消息推送

# evelopment Setup

```shell
1. cd pest-message-board-front/ 
2. npm run build
3. cd ..
4. cd pest-message-board-backend
5. mvn clean package
6. mv pest-message-board-backend-0.0.1-SNAPSHOT.jar ../
7. cd ..
8. docker-compose up
```
